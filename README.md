# Koditos Arcade

Proyecto para crear juegos sencillos estilo arcade/Atari.

## Estructura

```text
games/
  RoboChip/
    index.html
    src/
      game.js
      styles.css
    assets/
```

## Como jugar

Abre `index.html` en tu navegador para ver el menu principal.

Cada juego vive en su propia carpeta dentro de `games/`.
